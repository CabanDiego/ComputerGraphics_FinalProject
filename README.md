# CS 4361 Final Project

## Project Overview
This project is a game inspired by the game "Super Monkey Ball" that was made within Three.js implementing Ammo.js with the addition of using external hardware to control the movement of the map.

## Technologies Used

Three.js

Ammo.js

CSS

## Steps to Run

1. Clone the repository

2. Install Live Server Extension

3. Right Click on "index.html"

4. Choose "Open with Live Server"

## How to play

1. Use arrow keys on keyboard to move the map

2. Use external hardware (Teensy 4.1 microcontroller with a BNO055 9-DoF IMU) to simulate map movement
